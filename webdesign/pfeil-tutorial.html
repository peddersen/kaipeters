Web Tutorial: Einklapp-Pfeile wie bei Wikipedia


Wikipedia gibt dem Nutzer in der Webapp (also der mobilen Webseite) die Möglichkeit einzelne Kapitel von Artikeln ein- oder auszuklappen. Um anzuzeigen, ob das Kapitel aktuell ein- oder ausgeklappt ist, werden kleine Pfeil-Grafiken genutzt. Diese stehen links neben den Überschriften. Wenn die Pfeile nach oben zeigen, ist das Kapitel ausgeklappt; nach oben dann logischerweise eingeklappt. Der Nutzer schließt oder öffnet ein Kapitel indem er einmal auf die Überschrift tippt. Ein schönes kleines Tool, das gerade auf kleinen Bildschirmen – wie von Smartphones - extrem praktisch ist. Wer in seiner Webapp viel Text verwendet, seine Nutzer aber nicht ewig scrollen lassen will, für den kann dieses kleine Tool ganz hilfreich sein. Mit einem kleinen Tutorial zeige ich euch, wie ihr diesen Effekt ganz einfach selbst nachbauen könnt.

Wer sich vorher schon mal angucken will, wie das Ganze am Ende aussieht, kann sich hier die Demo angucken:

Zuerst mal das Markup: 

<body>
    <article>
        <section>
            <button/> Kapitel 1</button>
                <p>Lorem ipsum... </p>
        </section>
			
        <section>
            <button/> Kapitel 2</button>
                <p>Lorem ipsum..</p>
        </section>
			
        <section>
            <button"/> Kapitel 3</button>
                <p>Lorem ipsum...</p>
        </section>
     </article>
</body>

Wir verwenden hier drei Kapitel mit entsprechend drei Kapitelüberschriften. Die Pfeile samt Überschrift werden in ein <button> Element geschrieben, damit es später egal ist, ob man auf den Pfeil oder die den Text in der Überschrift tippt, um das Kapitel ein- oder auszuklappen. Der eigentliche Text des Kapitels steht klassisch im <p> Element und das ganze Konstrukt wird von den in HTML5 neuen Elementen <article> und <section> zusammengehalten. Jetzt packen wir die Pfeil Grafiken dazu. Die könnt ihr entweder direkt aus Wikipedia downloaden oder selber bauen. Innerhalb der Button-Tags aber vor dem Text für die Überschrift werden jetzt die Grafiken des nach unten zeigenden Pfeils und des nach oben zeigenden Pfeils eingefügt. Die Reihenfolge spielt dabei keine Rolle.

Grafiken einfügen:

                Kapitel 3
            </button>
                <p>Lorem ipsum...</p>

Jetzt geht’s ans CSS. Um einen einigermaßen Wikipedia-like Effekt hinzukriegen und um das Design schön clean zu halten, schlachten wir erstmal die Buttons soweit aus, dass wir nur noch schwarzen Text auf weißem Hintergrund sehen. Dafür weisen wir den Buttons im Markup eine Klasse zu

Buttons Klassen zuweisen:

<body>
    <article>
        <section>
            < button class="buttons_style">
            <img src="img/arrow-down.png"/><img src="img/arrow-up.png"/> Kapitel 1</button>
                <p>Lorem ipsum... </p>
        </section>
			
        <section>
            <button class="buttons_style">
            <img src="img/arrow-down.png"/><img src="img/arrow-up.png"/> Kapitel 2</button>
                <p>Lorem ipsum..</p>
        </section>
			
        <section>
            <button class="buttons_style">
            <img src="img/arrow-down.png"/><img src="img/arrow-up.png"/> Kapitel 3</button>
                <p>Lorem ipsum...</p>
        </section>
     </article>
</body>

und schrauben dann im Stylesheet den Standard-Hintergrund, den Rand und die Umrandung beim Klicken ab. Außerdem wird die Schriftgröße auf 20 Pixel gesetzt, damit das Ganze etwas größer wird.

Standard-Gestaltung der Buttons entfernen:

.buttons_style {
	background:none; 
	border:none; 
	outline-style:none; 
	font-size:20px;
}

Bis jetzt seht ihr im Browser beide Pfeile vor der Überschrift und ausgeklappten Text. Aber damit das Tool auch Sinn macht sollte natürlich nur ein Pfeil angezeigt werden und der Text eingeklappt sein, wenn eure Nutzer auf die Seite kommen. Zum ein- und ausklappen kommen wir gleich noch. Zunächst muss ein Pfeil verschwinden – und zwar der, der nach oben zeigt. Dazu weisen wir den Grafiken im HTML Code eine ID zu

Grafiken IDs zuweisen:

<body>
    <article>
        <section>
            <button class="buttons_style">
            <img id="pfeil_runter_1" src="img/arrow-down.png"/>
            <img id="pfeil_hoch_1" src="img/arrow-up.png"/> Kapitel 1</button>
                <p>Lorem ipsum... </p>
        </section>
			
        <section>
            <button class="buttons_style">
            <img id="pfeil_runter_2" src="img/arrow-down.png"/>
            <img id="pfeil_hoch_2" src="img/arrow-up.png"/> Kapitel 2</button>
                <p>Lorem ipsum..</p>
        </section>
			
        <section>
            <button class="buttons_style">
            <img id="pfeil_runter_3" src="img/arrow-down.png"/>
            <img id="pfeil_hoch_3" src="img/arrow-up.png"/> Kapitel 3</button>
                <p>Lorem ipsum...</p>
        </section>
     </article>
</body>

und lassen den nach oben zeigenden Pfeilen per CSS verschwinden. Auch der Text soll standardmäßig nicht angezeigt werden. Auch hier wird die Anzeige quasi unsichtbar gemacht:

Nach-oben-Pfeile und Absätze verstecken:

#pfeil_hoch_1, #pfeil_hoch_2, #pfeil_hoch_3 {
	display:none;
	}
	
p {
	display:none;
}

Kurzer Recap: Wenn der Pfeil nach oben zeigt ist das Kapitel ausgeklappt. Also sollen standardmäßig die nach unten zeigenden Pfeile zu sehen sein (bei eingeklappten Text). Das Switchen der Pfeile regeln wird gleich mit etwas Javascript. Hier wird zepto.js genutzt. Eine kleine Javascript Bibliothek, die extra wenig Speicher verbraucht und daher gerade für Webapps optimal ist. Die Bibliothek kann unter zeptojs.com heruntergeladen werden. Der JavaScript Code wird am besten vor dem schließenden <body> Tag eingefügt:

Zepto: JavaScript-Bibliothek für Webapps:

<script src="js/zepto.js"></script>
    <script>
        $(function(){

            $('#but_1').on('click', function(){
              $('#p_1').toggle();
            });
				
            $('#but_2').on('click', function(){
              $('#p_2').toggle();
            });
				
            $('#but_3').on('click', function(){
              $('#p_3').toggle();
            });
				
            $('#but_1').on('click', function(){
              $('#pfeil_hoch_1').toggle();
              $('#pfeil_runter_1').toggle();
            })
				
            $('#but_2').on('click', function(){
              $('#pfeil_hoch_2').toggle();
              $('#pfeil_runter_2').toggle();
            })
				
            $('#but_3').on('click', function(){
              $('#pfeil_runter_3').toggle();
              $('#pfeil_hoch_3').toggle();
            })
        });
    </script>

Nachdem die Zepto Bibliothek in der ersten Zeile eingebunden wurde, wird durch die ersten drei Blöcke bestimmt, dass beim Klicken der Überschriften, der Text angezeigt wird. Die Toggle-Funktion (dt.: umschalten) ist das, was wir hier brauchen. Sie sorgt dafür, dass ausgewählte Elemente versteckt werden und beim erneuten Aufruf wieder erscheinen. Wenn das Element, was mit der Toggle Funktion verknüpft ist, am Anfang schon versteckt ist, wird es logischerweise beim ersten Aufruf angezeigt. Durch diesen kleinen Trick gelingt auch der Wechsel zwischen den Pfeilen.
